# @arabcode/web

The [arabcode](https://github.com/abdallhx2/arabcode) website: an Astro 5 + Starlight docs site with an Arabic-first lander and the share-conversation viewer. The gold token system (`--ac-*` brand variables, dark/light) lives in `src/styles/custom.css`.

## Commands

| Command              | Action                                                        |
| :------------------- | :------------------------------------------------------------ |
| `bun install`        | Install dependencies (run at the repo root)                   |
| `bun run dev`        | Start the dev server at `http://localhost:4321/docs/`         |
| `bun run dev:remote` | Dev server with live share data (`VITE_API_URL` pre-set)      |
| `bun run build`      | Build the production site to `./dist/`                        |
