arabcode — الذكاء الاصطناعي في طرفيتك، بالعربية.

أول وكيل ذكاء اصطناعي للطرفية بعرض عربي كامل — اتجاه RTL سليم وحروف متصلة وتشكيل صحيح. تكتب سؤالك بالعربية، ويقرأ الوكيل الكود ويعدّله وتقرأ جوابه بالعربية.

ابدأ الآن التوثيق الكامل
npm i -g arabcode
جلسة حقيقية

هكذا تبدو البرمجة بالعربية

تكتب بالعربية، والوكيل يقرأ مشروعك ويعدّل الكود ويردّ عليك بالعربية.

arabcode — ~/my-app

# افتح مشروعك وشغّل الأداة

~/my-app $ arabcode

أنت أضف صفحة src/pages/about.tsx تعرض معلومات الفريق من ملف team.json، واربطها بزر في شريط التنقل العلوي.
arabcode سأنشئ صفحة about مع تحميل بيانات الفريق. هذا ما سأعدّله في src/router.tsx وsrc/components/Nav.tsx:
src/router.tsx +4
12import { Home } from "./pages/home";
13import { About } from "./pages/about";
14
15export const routes = [
16 { path: "/", component: Home },
17 { path: "/about", component: About },
18];
arabcode أنشأت src/pages/about.tsx وعدّلت router.tsx وNav.tsx. شغّل npm run dev وستجد الرابط جاهزًا.

عرض RTL وتشكيل عربي

اتجاه من اليمين لليسار وحروف متصلة بشكلها الصحيح داخل أي طرفية، مع بقاء الكود لاتينيًا أحادي العرض.

محرك مفتوح المصدر

مبني على opencode برخصة MIT — كود مفتوح كاملًا، والطبقة العربية هي إسهام arabcode.

كل المنصات

ماك ولينكس وويندوز — npm و Homebrew و curl و PowerShell، وكلها مدعومة دعمًا كاملًا.

التثبيت

ثبّتها في أقل من دقيقة

اختر منصتك، أو استخدم npm. الروابط المذكورة تُستبدل بعناوين النشر الفعلية.

سكربت التثبيت

يعمل على Apple Silicon و Intel. لا يتطلب صلاحيات جذر.

curl -fsSL https://raw.githubusercontent.com/abdallhx2/arabcode/main/install | bash

أو عبر Homebrew

brew install abdallhx2/tap/arabcode

سكربت التثبيت

يدعم x64 و arm64، ويكتشف glibc/musl تلقائيًا.

curl -fsSL https://raw.githubusercontent.com/abdallhx2/arabcode/main/install | bash

PowerShell

يُثبَّت إلى %LOCALAPPDATA%\arabcode\bin. شغّل في PowerShell.

irm https://raw.githubusercontent.com/abdallhx2/arabcode/main/install.ps1 | iex

npm — لكل المنصات

إن كان Node.js مثبّتًا، هذه أبسط طريقة على أي نظام.

npm i -g arabcode

من المصدر

للبناء المحلي بواسطة Bun.

git clone https://github.com/abdallhx2/arabcode
# بعد التثبيت — ابدأ خلال لحظات cd <مشروعك> arabcode
