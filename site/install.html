دليل التثبيت

ثبّت arabcode في دقيقة

يعمل على ماك ولينكس وويندوز (ويندوز و PowerShell مدعومان دعمًا كاملًا). اختر منصّتك أدناه، ثم تحقّق من التثبيت وابدأ.

المتطلبات

طرفية تدعم Unicode

ضرورية لعرض النص العربي مع التشكيل واتجاه RTL بشكل صحيح — مثل WezTerm أو Alacritty أو Ghostty أو Kitty.

Node.js و npm

مطلوبة فقط لطريقة التثبيت عبر npm. يُنصح بإصدار حديث ومدعوم من Node.js.

ripgrep اختياري

يُسرّع البحث داخل المشاريع بشكل ملحوظ، لكنه اختياري وليس شرطًا للتشغيل.

التثبيت حسب المنصّة

أوامر التثبيت الرسمية لـ arabcode — انسخها كما هي.

سكربت curl (موصى به)

curl -fsSL https://raw.githubusercontent.com/abdallhx2/arabcode/main/install | bash

Homebrew

brew install abdallhx2/tap/arabcode

يُثبَّت التنفيذي في $HOME/.arabcode/bin. تأكّد أن هذا المسار ضمن PATH.

سكربت curl (موصى به)

curl -fsSL https://raw.githubusercontent.com/abdallhx2/arabcode/main/install | bash

Homebrew (Linuxbrew)

brew install abdallhx2/tap/arabcode

يُثبَّت التنفيذي في $HOME/.arabcode/bin. تأكّد أن هذا المسار ضمن PATH.

PowerShell (موصى به)

irm https://raw.githubusercontent.com/abdallhx2/arabcode/main/install.ps1 | iex

npm

npm i -g arabcode

يُثبَّت التنفيذي في %LOCALAPPDATA%\arabcode\bin. للحصول على أفضل تجربة على ويندوز يُنصح باستخدام WSL. arabcode يدعم ويندوز و PowerShell دعمًا كاملًا.

npm (كل الأنظمة)

npm i -g arabcode

تعمل هذه الطريقة على ماك ولينكس وويندوز. تتطلّب Node.js و npm مثبّتَين مسبقًا.

من المصدر

bash
git clone https://github.com/abdallhx2/arabcode \
  && cd arabcode \
  && bun install \
  && bun run --cwd packages/opencode dev

يتطلّب Bun مثبّتًا. اسم مجلد الحزمة الداخلي packages/opencode موروث ولا يتغيّر.

التحقق من التثبيت

اعرض رقم الإصدار ثم شغّل الأداة:

bash
arabcode --version
arabcode

البدء السريع

افتح مجلد مشروعك وشغّل الأداة:

bash
cd <مشروعك>   # افتح مجلد المشروع
arabcode       # شغّل الأداة

بعد التشغيل، اربط موديلًا من صفحة النماذج والموديلات — ابدأ مجانًا أو اربط مفتاحك الخاص.

إزالة التثبيت

اختر الطريقة المطابقة لكيفية تثبيتك.

npm

npm uninstall -g arabcode

ماك / لينكس (curl أو من المصدر)

احذف مجلد التثبيت ثم أزِل مدخل PATH المرتبط به من ملف إعداد طرفيتك (~/.bashrc أو ~/.zshrc):

rm -rf ~/.arabcode

ويندوز (PowerShell)

احذف مجلد التثبيت ثم أزِل %LOCALAPPDATA%\arabcode\bin من متغيّر البيئة PATH:

Remove-Item -Recurse -Force "$env:LOCALAPPDATA\arabcode"

Homebrew

brew uninstall arabcode

الرئيسية النماذج الميزات التثبيت الصفحة التعريفية GitHub

arabcode يعرّب الطرفية بالكامل (اتجاه RTL + تشكيل عربي) — وهي إضافته الجوهرية فوق المحرّك. وهو مبني على opencode مفتوح المصدر — رخصة MIT. الميزات الموروثة هي من opencode؛ إسهام arabcode الأساسي هو طبقة العربية (الاتجاه والتشكيل والإدخال).
