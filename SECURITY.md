# الأمان

## مهم

لا نقبل تقارير أمان مُولَّدة بالذكاء الاصطناعي. نستقبل عددًا كبيرًا منها ولا نملك الموارد لمراجعتها جميعًا، وإرسال تقرير من هذا النوع يعرّضك للحظر من المشروع.

## نموذج التهديد

### نظرة عامة

arabcode مساعد برمجة يعمل بالذكاء الاصطناعي محليًا على جهازك. يوفّر نظام وكلاء يمتلك صلاحية الوصول إلى أدوات قوية تشمل تنفيذ الأوامر، وعمليات الملفات، والوصول إلى الويب.

### لا يوجد Sandbox

**لا** يعزل arabcode الوكيل داخل بيئة معزولة (sandbox). نظام الصلاحيات موجود كميزة تجربة استخدام لمساعدتك على إدراك ما يفعله الوكيل — فيطلب التأكيد قبل تنفيذ الأوامر وكتابة الملفات وغيرها — لكنه غير مصمَّم لتوفير عزل أمني.

إن احتجت عزلًا حقيقيًا، فشغّل arabcode داخل حاوية Docker أو جهاز افتراضي.

### وضع الخادم

وضع الخادم اختياري (opt-in). عند تفعيله، اضبط `OPENCODE_SERVER_PASSWORD` لطلب مصادقة HTTP Basic. بدونه يعمل الخادم دون مصادقة (مع تحذير)، ومسؤولية تأمين الخادم تقع على المستخدم النهائي؛ أي وظيفة يوفّرها الخادم ليست ثغرة.

### خارج النطاق

| الفئة                             | السبب                                                          |
| --------------------------------- | -------------------------------------------------------------- |
| **الوصول للخادم عند تفعيله**       | إذا فعّلت وضع الخادم فوصول الـ API سلوك متوقّع                    |
| **تجاوز الـ Sandbox**             | نظام الصلاحيات ليس sandbox (انظر أعلاه)                          |
| **معالجة بيانات مزوّد النموذج**    | البيانات المُرسَلة إلى مزوّد النموذج تحكمها سياسات المزوّد         |
| **سلوك خوادم MCP**                | خوادم MCP الخارجية التي تضبطها خارج حدود الثقة                    |
| **ملفات الإعداد الخبيثة**          | المستخدم يتحكّم بإعداداته؛ تعديلها ليس ناقلًا للهجوم              |

# الإبلاغ عن مشكلات الأمان

نقدّر جهودك في الإفصاح المسؤول عن اكتشافاتك، وسنبذل ما بوسعنا لتقدير مساهمتك.

للإبلاغ عن مشكلة أمنية، استخدم تبويب GitHub Security Advisory عبر ["الإبلاغ عن ثغرة"](https://github.com/abdallhx2/arabcode/security/advisories/new).

سيرسل الفريق ردًّا يوضّح الخطوات التالية للتعامل مع بلاغك، وسيبقيك على اطّلاع بتقدّم الإصلاح، وقد يطلب معلومات إضافية.
